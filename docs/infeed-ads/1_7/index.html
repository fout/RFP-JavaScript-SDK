AMP への対応

広告掲載ページが AMP の場合、以下のような記述で広告を埋め込むことができます。

<amp-ad
  type="rfp"
  width="320"
  height="250"
  data-adspot-id="XXXXXXXX"
  data-stylesheet-url="https://example.com/styles/ad.css">
</amp-ad>

各属性の説明

属性名 説明
type rfp 固定となり必須です。
width 広告ユニットの幅
height 広告ユニットの高さ
data-* 追加属性。必須のものと任意のものがあります。

data 属性で使用可能なパラメータ

パラメータ名 必須 説明 例
data-adspot-id Ad Spot ID XXXXXX
data-stylesheet-url 広告に適用するスタイルシートの URL https://example.com/styles/ad.css
